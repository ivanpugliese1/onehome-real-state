# Web Inmobiliaria

## Stack
- HTML5, CSS3, JavaScript vanilla
- Sin bundler, sin frameworks

## Estructura
- Ver @.claude/estructura.md

## Convenciones
- BEM para clases CSS
- camelCase para variables JS
- Un archivo JS por funcionalidad
- Imágenes optimizadas antes de subir (máx 200kb)

## SEO
- Cada página con title y meta description únicos
- Imágenes siempre con alt descriptivo y relevante
- URLs amigables: /propiedades/casa-3-ambientes-palermo
- Datos estructurados schema.org para propiedades (RealEstateListing)
- Meta og: tags para compartir en redes sociales
- Sitemap.xml y robots.txt configurados

## Validación
- Validar formularios en frontend antes de enviar
- Mostrar mensajes de error claros en español
- Campos obligatorios: nombre, email, teléfono, mensaje
- Validar formato de email y teléfono argentino

## Contexto
- Sitio estático para inmobiliaria argentina
- Público objetivo: compradores y vendedores de propiedades en Argentina
- Proyecto real, sera cargado en Hostinger.

## Tips
- Priorizar performance y carga rápida
- Diseño responsive mobile-first
- Compatibilidad con navegadores modernos
